## Priority Fix Order

1. **BUG-001 (XSS)** - Ship-stopper. Every user message is a potential attack vector.
2. **BUG-002 (ReferenceError)** - Blocks all non-LLM fallback paths.
3. **BUG-005 (TTS orphan)** - User-facing broken state on common browser configurations.
4. **BUG-004 (Race condition)** - Causes incorrect badge rendering under normal usage patterns.
5. **BUG-008 (Stuck typing)** - Leaves permanent broken UI state after back navigation.
6. **BUG-006 (Nested scroll)** - Degrades mobile UX significantly.
7. **BUG-010 (Mobile toggle)** - Missing feature that causes cramped mobile layout.
8. **BUG-007 (Modal overlay)** - Minor UX annoyance.
9. **BUG-009 (Duplicate API calls)** - Performance waste.
10. **BUG-003 (DOM rewrite)** - Future-proofing for event listener safety.
11. **BUG-011 through BUG-017** - Low priority, fix opportunistically.
